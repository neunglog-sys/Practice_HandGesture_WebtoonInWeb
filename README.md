# 🚀 Webtoon Hands-free Controller (AI 손동작 제어)

이 프로젝트는 **MediaPipe**와 **TensorFlow**를 활용하여 웹캠으로 사용자의 손동작을 인식하고, 네이버 웹툰의 인터페이스를 제어(스크롤 및 페이지 이동)할 수 있도록 만든 AI 컨트롤러입니다. 데이터 수집부터 모델 학습, 실제 구현까지 전체 파이프라인을 구축하였습니다.

---

## 🛠 1. 개발 환경 및 요구사항
- **Python**: 3.11
- **Main Libraries**:
  - `mediapipe == 0.10.11`
  - `tensorflow == 2.15.0`
  - `opencv-python`
  - `pyautogui`
  - `numpy`
  - `scikit-learn` (데이터 전처리용)
- **Editor**: VS Code, Jupyter Notebook (`.ipynb`)
- **Hardware**: 웹캠(Webcam) 필수
- **Target Browser**: Google Chrome (PC 웹 버전 네이버 웹툰 기준 최적화)

> **⚠️ 주의사항**: 본 프로그램은 개발자의 PC 해상도 및 브라우저 환경에 맞춰 마우스 좌표가 설정되어 있습니다. 사용자의 환경에 따라 동작이 정확하지 않을 수 있으며, 이 경우 아래 **[기타 도구]** 섹션을 참고하여 좌표를 수정해야 합니다.

---

## 📂 2. 사용 방법 (Workflow)

### **Step 1. 데이터셋 수집 (`create_dataset_webtoon.py`)**
학습에 필요한 손가락 관절(Landmark) 데이터를 수집합니다.
- 데이터는 `dataset/` 폴더 내에 `.npy` 형식으로 저장됩니다.
- **구성 동작**: `scroll_down`, `scroll_up`, `next_ep`, `prev_ep`, `menu`

### **Step 2. 모델 학습 (`train_webtoon.ipynb`)**
수집된 데이터를 바탕으로 딥러닝 모델을 학습시킵니다.
- `scikit-learn`을 사용하여 데이터를 분리하고 전처리를 수행합니다.
- 학습이 완료되면 `models/` 폴더 내에 `webtoon_model.h5` 파일이 생성됩니다.

### **Step 3. 프로그램 실행 (`test_webtoon.py`)**
실제로 웹툰을 보며 손동작으로 제어합니다.
- 카메라 화면 상단에 현재 인식 중인 동작과 확신도(%)가 실시간으로 표시됩니다.
- **클릭 동작(다음/이전/목록)** 이후에는 자동으로 대기 좌표로 마우스가 이동하며, 중복 실행 방지를 위해 스크롤 동작을 한 번 수행해야 다시 클릭 기능이 활성화됩니다.

---

## 🛠 3. 기타 도구 및 설정 수정
- **finger_pos.py**: 현재 마우스 커서의 좌표를 확인하는 도구입니다.
- **좌표 수정**: 사용자의 해상도나 브라우저 크기가 다를 경우, `finger_pos.py`로 다음/이전/목록 버튼의 좌표를 확인한 뒤 `test_webtoon.py` 코드 내의 `target` 좌표 값을 본인 환경에 맞게 수정하십시오. (기본 대기 좌표: `X=1598, Y=433`)

---

## 💡 주요 기능 특징
- **Always on Top**: 웹캠 창이 항상 브라우저 위에 떠 있도록 설정되어 있어 제어 상태를 실시간으로 모니터링할 수 있습니다.
- **동작 안정성**: 오인식을 방지하기 위해 특정 동작을 일정 시간 유지해야 실행되도록 설계되었으며, 텐서플로우 로그를 최적화하여 터미널에서 인식 상태를 깔끔하게 확인할 수 있습니다.
